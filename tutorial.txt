=== SushiConverter Tutorial ===

*** MANDATORY: Environment Setup ***
Setting up a dedicated environment is REQUIRED for stability.
This program is validated only with the specific versions in requirements.txt.

Installation steps:
1. Create a virtual environment (conda or venv)
2. Install dependencies:
   pip install -r requirements.txt

--- Usage Examples ---

1. Darknet to ONNX
python main.py --mode darknet --graph model.cfg --weights model.weights --output model.onnx --validate

2. PyTorch to ONNX
python main.py --mode pytorch --weights model.pt --output model.onnx --validate

3. Darknet to PyTorch (.pt)
python main.py --mode darknet --graph model.cfg --weights model.weights --output-mode pytorch

4. Darknet to Caffe
python main.py --mode darknet --graph model.cfg --weights model.weights --output-mode caffe

5. Standalone Python Source (.py + .pth)
python main.py --mode darknet --graph model.cfg --weights model.weights --output-mode source --output standalone_model

6. Weights Only (.pth)
python main.py --mode darknet --graph model.cfg --weights model.weights --output-mode pth

Notes:
- Use --validate to ensure the ONNX model matches the original.
- Use --no-simplify to disable ONNX model simplification.
- Use --yolo-layer to include decoded YOLO layers (default is raw tensors).
- Use --output-mode pytorch to save the model as a PyTorch file.
- Default export mode is ONNX if not specified.
- Default shape is 1x3x416x416 if not specified.
- If exporting without weights, skip the --weights flag; weights will be generated randomly.
- Opset 11 is enforced on ONNX models for compatibility.
